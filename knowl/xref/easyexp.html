Proposition 1.2.7.

For any \(\phi, \phi_1, \phi_2 \in \R\text{,}\)
  1. \(\displaystyle e^{ i \phi_1 } \, e^{ i \phi_2 } = e^{ i ( \phi_1 + \phi_2 ) }\)
  2. \(\displaystyle e^{ i 0 } = 1\)
  3. \(\displaystyle \frac 1 {e^{ i \phi }} = e^{ - i \phi }\)
  4. \(\displaystyle e^{ i ( \phi + 2 \pi ) } = e^{ i \phi }\)
  5. \(\displaystyle \left| e^{ i \phi } \right| = 1\)
  6. \(\frac{ d }{ \diff{\phi} } \, e^{ i \phi } = i \, e^{ i \phi }\text{.}\)
