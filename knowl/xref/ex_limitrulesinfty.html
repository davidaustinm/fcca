Exercise 11.

Show that our definition of \(\Chat\) honors the “finite” limit rules in Proposition 2.1.6, by proving the following, where \(a \in \C\text{:}\)
  1. If \(\lim_{z\to z_0}f(z)=\infty\) and \(\lim_{z\to z_0}g(z)=a\) then \(\lim_{z\to z_0} (f(z) + g(z)) = \infty \text{.}\)
  2. If \(\lim_{z\to z_0}f(z)=\infty\) and \(\lim_{z\to z_0}g(z)=a \neq 0\) then \(\lim_{z\to z_0} (f(z) \cdot g(z)) = \infty \text{.}\)
  3. If \(\lim_{z\to z_0}f(z)=\lim_{z\to z_0}g(z)=\infty\) then \(\lim_{z\to z_0} (f(z) \cdot g(z)) =\infty \text{.}\)
  4. If \(\lim_{z\to z_0}f(z)=\infty\) and \(\lim_{z\to z_0}g(z)=a\) then \(\lim_{z\to z_0} \frac{ g(z) }{ f(z) } = 0 \text{.}\)
  5. If \(\lim_{z\to z_0}f(z)=0\) and \(\lim_{z\to z_0}g(z)=a \ne 0\) then \(\lim_{z\to z_0} \frac{ g(z) }{ f(z) } = \infty \text{.}\)
