Example 4.4.3.

We’d like to determine
\begin{equation*} \int_{ C[i,1] } \frac{ \diff{z} }{ z^2 + 1 } \,\text{.} \end{equation*}
The function \(f(z) = \frac{ 1 }{ z+i }\) is holomorphic in \(\C \setminus \{ -i \}\text{,}\) which contains \(\overline D[i,1]\text{.}\) Thus we can apply Theorem 4.4.1:
\begin{equation*} \int_{ C[i,1] } \frac{ \diff{z} }{ z^2 + 1 } \= \int_{ C[i,1] } \frac{ \frac{ 1 }{ z+i } }{ z-i } \, \diff{z} \= 2 \pi i \, f(i) \= 2 \pi i \, \frac{ 1 }{ 2i } \= \pi \, \text{.} \end{equation*}
