Proposition 2.2.5.

Suppose \(f\) and \(g\) are differentiable at \(z \in \C\) and \(h\) is differentiable at \(g(z)\text{.}\) Then
  1. \(\bigl( f(z) + c \, g(z) \bigr)' \= f'(z) + c \, g'(z)\) for any \(c \in \C\)
  2. \(\displaystyle \bigl( f(z) \, g(z) \bigr)' \= f'(z) \, g(z) + f(z) \, g'(z)\)
  3. \(\displaystyle \left( \frac{ f(z) }{ g(z) } \right)' \= \frac{ f'(z) \, g(z) - f(z) \, g'(z) }{ g(z)^2 }\) provided that \(g(z)^2 \ne 0\)
  4. \(\bigl( z^n \bigr)' \= n \, z^{n-1}\) for any nonzero integer \(n\)
  5. \(g\) is continuous at \(z\)
  6. \(\bigl( h(g(z)) \bigr)' \= h'(g(z)) \, g'(z) \text{.}\)

Proof.

\begin{align*} \bigl( f(z) g(z) \bigr)' \amp \= \lim_{ h \to 0 } \frac{ f(z+h) \, g(z+h) - f(z) \, g(z) }{ h }\\ \amp \= \lim_{ h \to 0 } \frac{ f(z+h) \left( g(z+h) - g(z) \right) + \left( f(z+h) - f(z) \right) g(z) }{ h }\\ \amp \= \lim_{ h \to 0 } f(z+h) \frac{ g(z+h) - g(z) } h + \lim_{ h \to 0 } \frac{ f(z+h) - f(z) } h g(z)\\ \amp \= f(z) \, g'(z) + f'(z) \, g(z) \,\text{.} \end{align*}
Note that we have used the definition of the derivative and Proposition 2.1.6 Item a & Item b (the addition and multiplication rules for limits).
