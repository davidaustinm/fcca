Example 9.2.5.

The integrand \(\frac{ \exp(z) }{ \sin(z) }\) in Example 8.3.10 has poles of order 1 at 0 and \(\pi\text{.}\) We thus compute
\begin{equation*} \Res_{ z=0 } \left( \frac{ \exp(z) }{ \sin(z) } \right) \= \lim_{ z \to 0 } \left( z \, \frac{ \exp(z) }{ \sin(z) } \right) \= \exp(0) \lim_{ z \to 0 } \frac{ z }{ \sin(z) } \= 1 \end{equation*}
and
\begin{align*} \Res_{ z=\pi } \left( \frac{ \exp(z) }{ \sin(z) } \right) \amp \= \lim_{ z \to \pi } \left( (z-\pi) \, \frac{ \exp(z) }{ \sin(z) } \right)\\ \amp \= \exp(\pi) \lim_{ z \to \pi } \frac{ z-\pi }{ \sin(z) }\\ \amp \= - e^\pi\text{,} \end{align*}
confirming our computations in Example 8.3.10.
