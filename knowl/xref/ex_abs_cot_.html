Exercise 36.

Let \(z=x+iy\) and show that

(a)

\(\abs{\sin z}^2 = \sin^2x+\sinh^2y = \cosh^2y-\cos^2x\)

(b)

\(\abs{\cos z}^2 = \cos^2x+\sinh^2y = \cosh^2y-\sin^2x\)

(c)

If \(\cos x = 0\) then
\begin{equation*} \abs{\cot z}^2 \= \frac{\cosh^2y-1}{\cosh^2y} \ \le \ 1\,\text{.} \end{equation*}

(d)

If \(\abs{y}\ge1\) then
\begin{equation*} \abs{\cot z}^2 \ \le \ \frac{\sinh^2y+1}{\sinh^2y} \= 1+\frac1{\sinh^2y} \ \le \ 1+\frac1{\sinh^21} \ \le \ 2\, \text{.} \end{equation*}
