Exercise 19.

  1. Suppose \(G\subseteq\C\) and \(f_n : G \to \C\) for \(n \ge 1\text{.}\) Suppose \((a_n)\) is a sequence in \(\R\) with \(\lim_{ n \to \infty } a_n = 0\) and, for each \(n \ge 1\text{,}\)
    \begin{equation*} |f_n(z)| \ \le \ a_n \qquad \text{ for all } z \in G \,\text{.} \end{equation*}
    Show that \((f_n)\) converges uniformly to the zero function in \(G\text{.}\)
  2. Re-prove the statement of Example 7.3.3 using part a).
