Exercise 1.

Find the length of the following paths:
  1. \(\gamma(t) = 3t+i\text{,}\) \(-1 \leq t \leq 1\)
  2. \(\gamma(t) = i + e^{i \pi t}\text{,}\) \(0 \leq t \leq 1\)
  3. \(\gamma(t) = i\sin(t)\text{,}\) \(-\pi \leq t \leq \pi\)
  4. \(\gamma(t) = t - i \, e^{ -it }\text{,}\) \(0 \leq t \leq 2 \pi\)
Draw pictures of each path and convince yourself that the lengths you computed are sensible. (The last path is a cycloid, the trace of a fixed point on a wheel as it makes one rotation.)
