The function \(h(t,s)\) is called a homotopy. For each fixed \(s\text{,}\) a homotopy \(h(t,s)\) is a path parametrized by \(t\text{,}\) and as \(s\) goes from \(0\) to \(1\text{,}\) these paths continuously transform from \(\gg_0\) to \(\gg_1\text{.}\) The last condition in (4.4) simply says that each of these paths is also closed.
