Definition 3.1.1.

A linear fractional transformation is a function of the form
\begin{equation*} f(z) \= \frac{ az+b }{ cz+d } \end{equation*}
where \(a,b,c,d \in \C\text{.}\) If \(ad-bc \not= 0\) then \(f\) is called a Möbius
