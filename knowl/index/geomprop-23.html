Definition 1.2.8.

A root of unity is a number of the form \(e^{ 2 \pi i \frac m n }\) for some integers \(m\) and \(n > 0\text{.}\) Equivalently (by ExerciseÂ 1.5.17), a root of unity is a complex number \(\zeta\) such that \(\zeta^n = 1\) for some positive integer \(n\text{.}\) In this case, we call \(\zeta\) an \(n\th\) root of unity. If \(n\) is the smallest positive integer with the property \(\zeta^n = 1\text{,}\) then \(\zeta\) is a primitive \(n\th\) root of unity.
