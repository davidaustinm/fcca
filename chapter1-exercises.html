    Exercises 1.5 Exercises

    1.

    Let \(z = 1+2i\) and \(w = 2-i\text{.}\) Compute the following:
    1. \(\displaystyle z + 3w\)
    2. \(\displaystyle \overline{w} - z\)
    3. \(\displaystyle z^3\)
    4. \(\displaystyle \Re(w^2 + w)\)
    5. \(\displaystyle z^2 + \overline{z} + i\)

    2.

    Find the real and imaginary parts of each of the following:
    1. \(\frac{z - a}{z + a}\) for any \(a \in \R\)
    2. \(\displaystyle \frac{3 + 5i}{7i + 1}\)
    3. \(\displaystyle \left( \frac{-1 + i\sqrt{3}}{2} \right)^3\)
    4. \(i^n\) for any \(n \in \Z\)

    3.

    Find the absolute value and conjugate of each of the following:
    1. \(\displaystyle -2 + i\)
    2. \(\displaystyle (2 + i)(4 + 3i)\)
    3. \(\displaystyle \frac{3 - i}{\sqrt{2} + 3i}\)
    4. \(\displaystyle (1 + i)^6\)

    4.

    Write in polar form:
    1. \(\displaystyle 2i\)
    2. \(\displaystyle 1+i\)
    3. \(\displaystyle -3 + \sqrt 3 \, i\)
    4. \(\displaystyle -i\)
    5. \(\displaystyle (2 - i)^2\)
    6. \(\displaystyle |3 - 4i|\)
    7. \(\displaystyle \sqrt{5} - i\)
    8. \(\displaystyle \left(\frac{1-i}{\sqrt{3}}\right)^4\)

    5.

    Write in rectangular form:
    1. \(\displaystyle \sqrt 2 \, e^{ i \frac{ 3 \pi } 4 }\)
    2. \(\displaystyle 34 \, e^{i \frac \pi 2}\)
    3. \(\displaystyle - e^{i 250 \pi}\)
    4. \(\displaystyle 2 \, e^{4\pi i}\)

    6.

    Write in both polar and rectangular form:
    1. \(\displaystyle e^{\ln(5)}i\)
    2. \(\displaystyle \frac{d}{\diff\phi} \, e^{\phi + i\phi}\)

    7.

    Show that the quadratic formula works. That is, for \(a,b,c \in \R\) with \(a \ne 0\text{,}\) prove that the roots of the equation \(az^2+bz+c = 0\) are
    \begin{equation*} \frac{-b\pm \sqrt{b^2-4ac}}{2a} \,\text{.} \end{equation*}
    Here we define \(\sqrt{b^2-4ac} = i \sqrt{-b^2+4ac}\) if the discriminant \(b^2-4ac\) is negative.

    8.

    Use the quadratic formula to solve the following equations.
    1. \(\displaystyle z^2 + 25 = 0\)
    2. \(\displaystyle 2z^2 + 2z + 5 = 0\)
    3. \(\displaystyle 5z^2 + 4z + 1 = 0\)
    4. \(\displaystyle z^2 - z = 1\)
    5. \(\displaystyle z^2 = 2z\)

    9.

    Find all solutions of the equation \(z^2 + 2 z + (1 - i) = 0\text{.}\)

    10.

    Fix \(a \in \C\) and \(b \in \R\text{.}\) Show that the equation \(|z^2| + \Re(az) + b = 0\) has a solution if and only if \(|a^2| \geq 4b\text{.}\) When solutions exist, show the solution set is a circle.

    11.

    Find all solutions to the following equations:
    1. \(\displaystyle z^6 = 1\)
    2. \(\displaystyle z^4 = -16\)
    3. \(\displaystyle z^6=-9\)
    4. \(\displaystyle z^6-z^3-2=0\)

    12.

    Show that \(|z| = 1\) if and only if \(\frac 1 z = \overline{z}\text{.}\)

    13.

    Show that
    1. \(z\) is a real number if and only if \(z = \o z\text{;}\)
    2. \(z\) is either real or purely imaginary if and only if \((\o{z})^2 = z^2\text{.}\)

    14.

    Prove Proposition 1.1.2.

    15.

    Show that if \(z_1 \, z_2 = 0\) then \(z_1 = 0\) or \(z_2 = 0\text{.}\)

    16.

    Prove Proposition 1.2.7.

    17.

    Fix a positive integer \(n\text{.}\) Prove that the solutions to the equation \(z^n = 1\) are precisely \(z = e^{ 2 \pi i \frac m n }\) where \(m \in \Z\text{.}\) (Hint: To show that every solution of \(z^n = 1\) is of this form, first prove that it must be of the form \(z = e^{ 2 \pi i \frac a n }\) for some \(a \in \R\text{,}\) then write \(a = m + b\) for some integer \(m\) and some real number \(0 \le b \lt 1\text{,}\) and then argue that \(b\) has to be zero.)

    18.

    Show that
    \begin{equation*} z^5 - 1 \= \left( z-1 \right) \left( z^2 + 2z \cos \tfrac{ \pi }{ 5 } + 1 \right) \left( z^2 - 2z \cos \tfrac{ 2 \pi }{ 5 } + 1 \right) \end{equation*}
    and deduce from this closed formulas for \(\cos \frac{ \pi }{ 5 }\) and \(\cos \frac{ 2 \pi }{ 5 }\text{.}\)

    19.

    Fix a positive integer \(n\) and a complex number \(w\text{.}\) Find all solutions to \(z^n = w\text{.}\) (Hint: Write \(w\) in terms of polar coordinates.)

    20.

    Use Proposition 1.2.7 to derive the triple angle formulas:
    1. \(\displaystyle \cos (3 \phi) = \cos^3 \phi - 3 \cos \phi \sin^2 \phi\)
    2. \(\displaystyle \sin (3 \phi) = 3 \cos^2 \phi \sin \phi - \sin^3 \phi\)

    21.

    Given \(x, y \in \R\text{,}\) define the matrix \(M(x,y) := \displaystyle\begin{bmatrix}x \amp -y \\ y \amp x \end{bmatrix}\text{.}\) Show that
    \begin{equation*} M(x,y) + M(a,b) \ = \ M(x+a, \, y+b) \qquad \text{ and } \qquad M(x,y) \, M(a,b) \ = \ M(xa-yb, \, xb+ya) \,\text{.} \end{equation*}
    (This means that the set \(\{ M(x,y) : \, x, y \in \R \}\text{,}\) equipped with the usual addition and multiplication of matrices, behaves exactly like \(\C = \{ (x,y) : \, x,y \in \R \}\text{.}\))

    22.

    Prove Proposition 1.3.3.

    23.

    Sketch the following sets in the complex plane:
    1. \(\displaystyle \left\{ z \in \C : \, \left| z - 1 + i \right| = 2 \right\}\)
    2. \(\displaystyle \left\{ z \in \C : \, \left| z - 1 + i \right| \leq 2 \right\}\)
    3. \(\displaystyle \left\{ z \in \C : \, \Re (z+2-2i) = 3 \right\}\)
    4. \(\displaystyle \left\{ z \in \C : \, \left| z - i \right| + \left| z + i \right| = 3 \right\}\)
    5. \(\displaystyle \left\{ z \in \C : \, |z| = |z + 1| \right \}\)
    6. \(\displaystyle \left\{ z \in \C : \, |z-1| = 2 \, |z + 1| \right \}\)
    7. \(\displaystyle \left\{ z \in \C : \, \Re(z^2) = 1 \right \}\)
    8. \(\displaystyle \left\{ z \in \C : \, \Im(z^2) = 1 \right \}\)

    24.

    Suppose \(p\) is a polynomial with real coefficients. Prove that
    1. \(\o{p(z)} = p \left( \o z \right)\text{.}\)
    2. \(p(z) = 0\) if and only if \(p \left( \o z \right) = 0\text{.}\)

    25.

    Prove the reverse triangle inequality (Item b of Corollary 1.3.5) \(\left| z_1 - z_2 \right| \geq \left| z_1 \right| - \left| z_2 \right| \text{.}\)

    26.

    Use the previous exercise to show that
    \begin{equation*} \left| \frac{1}{z^2 - 1} \right| \leq \frac{1}{3} \end{equation*}
    for every \(z\) on the circle \(C[0,2]\text{.}\)

    27.

    Sketch the sets defined by the following constraints and determine whether they are open, closed, or neither; bounded; connected.
    1. \(\displaystyle \abs{z+3}\lt 2\)
    2. \(\displaystyle \abs{\Im(z)}\lt 1\)
    3. \(\displaystyle 0\lt \abs{z-1}\lt 2\)
    4. \(\displaystyle \abs{z-1}+\abs{z+1}=2\)
    5. \(\displaystyle \abs{z-1}+\abs{z+1}\lt 3\)
    6. \(\displaystyle \abs{z} \ge \Re(z) + 1\)

    28.

    What are the boundaries of the sets in the previous exercise?

    29.

    Let \(G\) be the set of points \(z \in \C\) satisfying either \(z\) is real and \(-2\lt z\lt -1\text{,}\) or \(|z|\lt 1\text{,}\) or \(z=1\) or \(z=2\text{.}\)
    1. Sketch the set \(G\text{,}\) being careful to indicate exactly the points that are in \(G\text{.}\)
    2. Determine the interior points of \(G\text{.}\)
    3. Determine the boundary points of \(G\text{.}\)
    4. Determine the isolated points of \(G\text{.}\)

    30.

    The set \(G\) in the previous exercise can be written in three different ways as the union of two disjoint nonempty separated subsets. Describe them, and in each case say briefly why the subsets are separated.

    31.

    Show that the union of two regions with nonempty intersection is itself a region.

    32.

    Show that if \(A \subseteq B\) and \(B\) is closed, then \(\partial A \subseteq B\text{.}\) Similarly, if \(A \subseteq B\) and \(A\) is open, show that \(A\) is contained in the interior of \(B\text{.}\)

    33.

    Find a parametrization for each of the following paths:
    1. the circle \(C[1+i, 1]\text{,}\) oriented counter-clockwise
    2. the line segment from \(-1-i\) to \(2i\)
    3. the top half of the circle \(C[0, 34]\text{,}\) oriented clockwise
    4. the rectangle with vertices \(\pm 1 \pm 2i\text{,}\) oriented counter-clockwise
    5. the ellipse \(\{ z \in \C : \, |z-1| + |z+1| = 4 \}\text{,}\) oriented counter-clockwise

    34.

    Draw the path parametrized by
    \begin{equation*} \gg(t) = \cos(t) \left| \cos(t) \right| + i \sin(t) \left| \sin(t) \right| , \qquad 0 \le t \le 2 \pi \,\text{.} \end{equation*}

    35.

    Let \(G\) be the annulus determined by the inequalities \(2\lt \abs z\lt 3\text{.}\) This is a connected open set. Find the maximum number of horizontal and vertical segments in \(G\) needed to connect two points of \(G\text{.}\)
